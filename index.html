How to add a new brief

  1. Put the brief file(s) in the repo (HTML/MD/CSV, images, etc.).
  2. Open Branding/BFIndex/briefs.json and add a new item.
  3. Commit & push to GitHub. Your links will work once GitHub Pages is updated.

Tip: For HTML previews, keep them self-contained (inline CSS) so external viewers can open them without build steps.
